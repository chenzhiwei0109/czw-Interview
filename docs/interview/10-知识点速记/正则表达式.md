## 栗子

数字千位分隔符表示

```js
var result = '12345678' .replace(/(?=\d{3}$)/g,',')
//=> '12345,6789'

//(?=(\d{3})+$)至少出现一次或者多次
var result = '123456789' .replace(/(?=(\d{3})+$)/g,',')
// ',123,456,789'

var result = '123456789'.replace(/(?!^)(?=(\d{3})+$)/);

```

## 读

```js
/^(\d{15}|\d{17}[\dxX])$ /
\d{15}:15位连续数字
\d{17}[\dxX]：17位连续数字+最后一位是数字或者xX
^$ :以这个开头并且结尾，仅有他
```
