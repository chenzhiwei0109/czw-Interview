## 贪心算法简介

- 是一种设计思想
- 期盼通过每个阶段的局部最优选择，而达到全局最优
- 结果不一定最优

零钱兑换

- 输出能够兑换总额的硬币的最少数量。
  
- 先选最大数目的硬币，
  
- ```js
  输入 coins = [1,2,5],amount = 11;
  输出:3
  解释:11=5+5+1
  ```

  ```js
  输入 coins [1,3,4],amount = 6;
  输出:3
  解释:6 = 4+1+1
  但是这不是最优解
  ```
