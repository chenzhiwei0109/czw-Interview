## 自定义组件实现双向绑定

- 需要在子组件处理@input事件

父组件

```html
<Vmodel v-model="data"></Vmodel>
<!-- 相当于
<Vmodel :value='course' @input='course=$event'></Vmodel>
-->
```

子组件

```html
<input type="text" :value="value" @input="$emit('input',$event.target.value)" />
<script>
    export default {
        props: ["value"],
    };
</script>
```
